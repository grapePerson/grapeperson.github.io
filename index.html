Dangerous trip

About

Genre

Dangerous trip - is a browser game in which the player character is continuously moving forward through the game world. He meets a lot of monsters on his way and battles with them by the spells.

Technologies

The following modern web technologies were used while building the game: JavaScript, Canvas, Konva.js, jQuery, Web Audio API, Lodash, Webpack.
The web architecture of the game is based on MVC design pattern.

Resolution

Resolution of the game screen is 1280 x 700, so if your device does not comply with this regulation feel free to close this site and forget it, or reduce the browser window (ctrl -).

Controls

The main player is controlled by keyboard. The countdown timer is used to motivate you to defeat as many monsters as possible.
